# YouTube Downloader

A simple web application built with HTML, CSS, JavaScript, Flask, and `yt-dlp` that allows users to download YouTube videos as MP3 or MP4 files.

## Features

* Download YouTube videos in MP4 format
* Extract audio and download as MP3
* Simple and responsive user interface
* Flask backend with `yt-dlp`

## Requirements

* Python 3
* Flask
* yt-dlp

Install the required packages:

```bash
pip install -r requirements.txt
```

## Running the Project

1. Start the Flask server:

```bash
python app.py
```

2. Open the application in your web browser.

3. Paste a YouTube URL.

4. Select the desired format (MP3 or MP4).

5. Click **Download**.
